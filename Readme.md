# 📇 PeopleDesk (Smart Contact Manager)

PeopleDesk is a Smart Contact Management web application built using Spring Boot.
It allows users to manage, organize, and interact with their contacts efficiently.


> ⚠️ **Note:** This project is currently under development.
---

## 🚀 Features

* User signup with email & password
* Email verification system
* Social login (Google & GitHub)
* Add contacts with profile picture
* Upload contact images to cloud (AWS / Cloudinary)
* View all contacts
* View detailed contact information
* Update and delete contacts
* Search contacts
* Pagination support
* Export contacts to Excel
* Mark contacts as favorite
* Send emails directly from the app (with attachments)
* User profile management
* Dark & Light theme support
* Feedback system


---

## 🛠️ Technologies Used

* Spring Boot
* Spring MVC
* Spring Data JPA (Hibernate ORM)
* Spring Security
* OAuth2 (Google & GitHub login)
* Thymeleaf
* Tailwind CSS + Flowbite
* JavaScript (Dark/Light mode)
* MySQL / PostgreSQL
* AWS / Cloudinary (File storage)
* Java Mail Service
* Bean Validation
* ModelMapper
* Lombok
* DevTools
